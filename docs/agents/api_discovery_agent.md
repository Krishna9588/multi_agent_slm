# Api Discovery Agent (api_discovery_agent.py)

## Brief Description
An API Integrator Agent. Use this to interact with external REST APIs dynamically.

## Prerequisites
1. **API Keys**: Keys for whatever external service you are attempting to contact.

## Step-by-Step Setup Guide
1. Identify the third-party service the agent needs to hit.
2. Export the appropriate token to your terminal environment before running the swarm.

## How to Update
- The code for this agent lives in `agents/api_discovery_agent.py`.
- To modify its behavior or add new parameters, edit the `PARAMETERS` dictionary and the primary function in the Python file.
